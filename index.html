Privacy Policy

This Privacy Policy governs the data practices of the ZippyMap: Get Help Nearby mobile application ("App"), which is solely operated by The Developer.

1. Developer and Contact Information

Developer: The Developer (Personal Account)
Contact for Privacy Questions: zippymap.team@gmail.com

2. Information Collected

A. Information Collected from All Users (Via Google Sign-In)
When you log in using Continue with Google, the Developer collects Full Name and Email Address for identification and account management.

B. Additional Information Collected from Service Providers Only
If a User registers as a Service Provider, we collect your Phone Number to allow Normal Users to contact you directly.

C. Location Data (Precise GPS Coordinates)
Location data is collected only for Service Providers when the "Live Location" button is actively enabled. This data is only processed while the App is in use to facilitate real-time discovery on the map.

3. Purpose of Data Collection

  • Account Authentication and identity verification.
  • Core Map Functionality: Allowing Normal Users to view Service Providers nearby.
  • Communication: Facilitating contact between users.

4. Third-Party Services

We use Google Play Services, Firebase Authentication, and Firestore for secure data processing and storage on Google's encrypted servers.
